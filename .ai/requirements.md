# Requirements

In order to run this project locally we need:

- Ruby (you can check the version in the `.ruby-version` file)
- Node.js (you can check in `.node-version` file)
- PostgreSQL (any recent version)
- Redis server (required for background jobs)
- Google Chrome (required for system tests using Capybara)

If any of these requirements is not installed, refer to the official installation guide: <https://docs.decidim.org/en/develop/install/manual>
