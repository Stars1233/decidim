= Close Debate

To use the API to close any debate that the user has created, you will need to use the following API request.

[source,graphql]
----
mutation closeDebate($componentId: ID!, $debateId: ID!, $input: CloseDebateInput!){
  component(id: $componentId) {
    ... on DebatesMutation {
      debate(id: $debateId) {
        close(input: $input) {
          id
          title {
            translation(locale: "en")
          }
          conclusions {
            translation(locale: "en")
          }
        }
      }
    }
  }
}
----

Example of submitted variables

[source,json]
----
{
  "componentId": "9",
  "debateId": "2",
  "input": {
    "locale": "en",
    "attributes": {
      "conclusions": "Your closing report"
    }
  }
}
----
