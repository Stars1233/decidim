= Delete Milestone

== Sample request

The `DeleteMilestone` mutation allows authenticated administrators to delete milestone within a result in a Decidim accountability component.

[source,graphql]
----
mutation deleteMilestone($componentId: ID!, $resultId: ID!, $milestoneId: ID!){
  component(id: $componentId){
    ... on AccountabilityMutation {
      result(id: $resultId){
        deleteMilestone(id: $milestoneId){
          id
        }
      }
    }
  }
}
----

Example of submitted variables

[source,json]
----
{
  "componentId": "3",
  "resultId": "5",
  "milestoneId": "10"
}
----
